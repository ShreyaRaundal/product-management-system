# product-management-system
Product listing with search and filter

# Product Management System

## Features
- Product listing
- Search products by name
- Filter by category
- Add new product

## Tech Stack
- Node.js
- Express.js
- MySQL
- HTML
- CSS
- JavaScript

## Setup

1 Install dependencies

npm install

2 Run server

node server.js

3 Open frontend

frontend/index.html
